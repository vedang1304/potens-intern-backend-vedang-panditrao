# Profile-to-Recommendation API

A backend REST API built using **Node.js, Express.js, PostgreSQL, and Prisma ORM** that recommends the top three job opportunities based on a candidate's profile.


---

# Features

## Core Features

- Profile-based recommendation engine
- Returns top 3 ranked job recommendations
- Detailed reasoning for every recommendation
- Complete CRUD operations for job catalogue
- Admin authentication using API token
- Eligibility explanation endpoint
- PostgreSQL database with Prisma ORM
- Seed script containing 15 job listings
- Input validation
- Unit tests

---

## Stretch Features Implemented

- Recommendation caching using Node Cache
- Swagger / OpenAPI documentation
- Docker & Docker Compose support

# Recommendation Logic

Each recommendation is generated by evaluating multiple profile attributes.

The matching score considers:

- Technical Skills
- Years of Experience
- Educational Qualification
- Preferred Job Category
- Preferred Work Mode
- Preferred Location
- Minimum Salary Expectation

We calculate the score based on these above listed requirements we have given these a specific scores so the particular scores are been calculated and according to it we fetch the recommendations of top 3 and provide it to the user.

# API Endpoints

## Recommendation

### POST /recommend

Returns the top three matching jobs.

Example Request

```json
{
    "skills":["Java","Spring Boot","SQL"],
    "experience":2,
    "education":"BTech",
    "preferredLocation":"Pune",
    "preferredWorkMode":"HYBRID",
    "preferredCategory":"Backend",
    "minimumSalary":1000000
}
```

---

## Job Catalogue

### GET /items

Returns all jobs.

Requires Admin Token.

---

### POST /items

Create a new job.

Requires Admin Token.

---

### PUT /items/:id

Update a job.

Requires Admin Token.

---

### DELETE /items/:id

Delete a job.

Requires Admin Token.

---

## Eligibility Explanation

### GET /explain/:item_id

Returns the eligibility criteria for a specific job.

Example Response

```json
{
    "success": true,
    "data": {
        "jobId": 2,
        "title": "Backend Developer",
        "eligibility": "Candidates should have 1-3 years of experience, a BTech degree, and skills in Java, Spring Boot and SQL."
    }
}
```

# Design Decisions

### Prisma ORM

Prisma was selected for its type safety, migration support, and simplified database access.

### PostgreSQL

PostgreSQL provides reliable relational storage suitable for structured recommendation data.

### Recommendation Scoring

Instead of strict filtering, a weighted scoring algorithm was implemented. This allows partially matching jobs to still be recommended if they satisfy important criteria.

### Caching

Recommendation responses are cached using Node Cache to improve performance for repeated requests with identical profiles.

### Swagger

Swagger was added to provide interactive API documentation for easier testing and integration.

### Docker

Docker Compose enables one-command setup of both the application and PostgreSQL database.
